# Configuration

Illagers Love Armor is configured through datapacks, just like the Forge mod.

I.L.A registers a new loot type - `illagerslovearmor:slot` - which is used to randomize the armor on the illager entity. 
The mod also registers the `illagerslovearmor:wave` condition which can be used to lock armor to a certain raid wave.

<Callout type="info">
More information on loot tables can be found [on the Minecraft Wiki](https://minecraft.fandom.com/wiki/Loot_table)
</Callout>

## Example

This example is used for the chestplate slot.

Diamond chestplates have the following chances:

- $4.5\%$ chance of having a diamond chestplate on 3
- Will have a diamond chestplate on wave $4$, $5$, $6$ and $7$ if the previous conditions/entries are not satisfied.

          "type": "minecraft:item",
